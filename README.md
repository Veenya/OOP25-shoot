# OOP25-shoot
ShOOts è un videogioco roguelike d’azione ispirato a Vampire Survivors, in cui il giocatore controlla un personaggio che si muove in un’arena mentre affronta ondate sempre più numerose di nemici, migliorando armi e abilità per sopravvivere il più a lungo possibile.

---

## How to:
Per far buildare il progetto usare:

```pws
.\gradlew.bat build
```

NON `gradle build`.
